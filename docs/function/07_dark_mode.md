# ダークモード機能

## 概要

画面の配色をダークテーマに切り替える機能。

## 切り替え方法

ヘッダー右側の🌙/☀️ボタンをクリック

## 永続化

- localStorageに設定を保存
- 次回アクセス時に自動で復元

## 対応コンポーネント

- ヘッダー
- 検索ボックス
- 入力フィールド
- タグフィルター
- カード（グリッド/リスト）
- 詳細ページ
- プロセス表示
- ページネーション

## カラーパレット

### ライトモード
- 背景: #667eea → #764ba2（グラデーション）
- カード背景: white
- テキスト: #333

### ダークモード
- 背景: #1a1a2e → #16213e（グラデーション）
- カード背景: #2a2a4a
- テキスト: #eee
