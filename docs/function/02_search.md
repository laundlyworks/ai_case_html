# 検索機能

## 概要

キーワードを入力して事例を絞り込む機能。

## 検索対象フィールド

- 事例番号
- タイトル
- サブタイトル
- 課題（プライマリ/セカンダリ）
- ソリューション（プライマリ/セカンダリ）
- 成果（プライマリ/セカンダリ）
- ユーザーの声
- タグ

## 動作

1. テキスト入力時にリアルタイムで絞り込み
2. 検索結果件数を表示
3. ヒットしたキーワードをハイライト表示

## オートコンプリート

- タグ一覧から候補を表示
- サブタイトルから候補を表示
- 上下キーで選択、Enterで決定
- 候補クリックでも選択可能

## キーボードショートカット

- `⌘/Ctrl + K`: 検索ボックスにフォーカス
