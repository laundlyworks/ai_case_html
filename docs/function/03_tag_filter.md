# タグフィルター機能

## 概要

タグをクリックして事例を絞り込む機能。

## 表示位置

検索ボックスの下部に全タグを表示。

## 動作

1. タグクリックで該当事例のみ表示
2. アクティブなタグはハイライト表示
3. 再クリックでフィルター解除
4. 検索クエリとの併用が可能

## タグ説明機能

タグをクリックすると、そのタグの説明がタグ一覧の下に表示される。

### データソース

- ファイル: `data/tags.js`
- 形式: `TAGS`配列（各要素に`tag`と`explanation`プロパティ）

### 動作詳細

1. タグをクリック → 説明がある場合、タグ一覧下部に表示
2. タグの選択解除 → 説明を非表示
3. 別のタグを選択 → 新しいタグの説明に切り替え
4. 説明がないタグ → 「説明はありません」と表示

### 表示スタイル

- グラデーション背景
- 左側にアクセントボーダー
- ダークモード対応

## タグの種類（例）

- NotebookLM
- Gemini
- ChatGPT
- Claude
- GitHub Copilot
- Dify
- 議事録
- 文書作成
- など
