# 詳細ページ機能

## 概要

事例カードをクリックすると表示される詳細情報ページ。

## 表示内容

1. **事例番号**: バッジ形式で表示
2. **タイトル**: 大きな見出しで表示
3. **スライド画像**: 元PDFのスライド画像を大きく表示
4. **プロセス（自動生成）**: 事例の実施手順をステップ形式で表示
5. **プロンプト例**: AIを使うステップに具体的なプロンプト例を表示

## プロンプト例表示

- AIを活用するステップにのみ表示
- 緑系のグラデーション背景で視覚的に区別
- 「💡 プロンプト例」ラベル付き
- モノスペースフォントで表示
- ダークモード対応

## 操作

- 「← 一覧に戻る」ボタンで一覧ページへ
- ヘッダーは常に表示

## レイアウト

- 最大幅: 1000px
- 中央寄せ
- 背景にグラデーション
