# ヘッダー機能

## 概要

画面上部に固定表示されるヘッダーコンポーネント。一覧ページ・詳細ページの両方で共通して表示される。

## 構成要素

### タイトル（左側）

- テキスト: 「AI活用100本ノック」
- クリックで一覧ページのトップへスクロール

### アクションボタン（右側）

1. **ダークモード切り替えボタン**
   - アイコン: 🌙（ライトモード時）/ ☀️（ダークモード時）
   - クリックでモード切り替え

2. **PDFリンクボタン**
   - アイコン: 📄
   - クリックで元のPDFファイルを新規タブで開く
   - URL: https://fullswing.dena.com/pdf/AI_100tips_slide.pdf

## スタイル

- 背景: 半透明白（ライト）/ 半透明ダーク（ダーク）
- backdrop-filter: blur(10px)
- 高さ: 60px
- z-index: 1000
