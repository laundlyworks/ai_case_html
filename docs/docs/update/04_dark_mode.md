# ダークモード実装

## 変更内容

### 機能追加

- ダークモード/ライトモード切り替え機能
- localStorageによる設定永続化
- ページ再読み込み時の設定復元

### 対応コンポーネント

- body背景
- ヘッダー
- 検索ボックス
- 入力フィールド・プレースホルダー
- オートコンプリートリスト
- タグフィルター
- 表示切り替えボタン
- カード（グリッド/リスト）
- 詳細ページ
- 詳細カード
- プロセス表示
- ページネーションボタン

### カラースキーム

- ダーク背景: #1a1a2e → #16213e
- カード背景: #2a2a4a
- テキスト: #eee / #ccc / #aaa
- アクセントカラー: #a0b4f4
