# プロンプト例の追加

## 変更内容

### process.jsの拡張

- 各プロセスステップに`prompt`フィールドを追加
- AIを活用するステップに具体的なプロンプト例を記載
- 全100件の事例に対応

### データ構造

```javascript
{
    "3.キーワードを用いたプロンプト入力": "説明文",
    "prompt": "この法律の第3条について、事業者が遵守すべき義務を箇条書きで整理してください"
}
```

### 詳細ページの表示

- プロンプト例がある場合、ステップ説明の下に表示
- 緑系グラデーション背景で視覚的に区別
- 「💡 プロンプト例」ラベル付き
- モノスペースフォントで可読性向上

### スタイル

- ライトモード: 緑系グラデーション背景
- ダークモード: 暗い緑系背景に対応
- 左側にアクセントボーダー

## 目的

- プロセスの具体性向上
- AIへの指示方法の参考例を提供
- 事例の再現性・活用性を高める
