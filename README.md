# AI活用100本ノック 検索アプリ

DeNA社が公開した「AI活用100本ノック」のスライドをインタラクティブに検索・閲覧できるWebアプリケーションです。

## 概要

このアプリケーションは、DeNA社内のAI活用事例100選を簡単に検索・閲覧するためのツールです。キーワード検索、タグフィルタリング、詳細表示などの機能を提供します。

## 機能

- **キーワード検索**: タイトル、課題、ソリューション、成果、タグなどから横断的に検索
- **オートコンプリート**: 入力中にタグやタイトルの候補を表示
- **タグフィルター**: タグをクリックして事例を絞り込み
- **表示切替**: グリッド表示とリスト表示の切り替え
- **詳細ページ**: 各事例のスライド画像とプロセス（プロンプト例含む）を表示
- **ダークモード**: ライト/ダークモードの切り替え対応
- **キーボードショートカット**: `Cmd/Ctrl + K` で検索ボックスにフォーカス

## ファイル構成

```
.
├── index.html          # メインのHTMLファイル（CSS/JS含む）
├── data/
│   ├── master.js       # 事例のマスターデータ
│   ├── process.js      # 各事例のプロセス・プロンプト例
│   ├── tags.js         # タグの説明情報
│   └── images/         # スライド画像（tips001.jpeg〜tips100.jpeg）
├── LICENSE.txt         # ライセンス情報
└── README.md
```

## 使い方

1. `index.html` をブラウザで開くだけで動作します（サーバー不要）
2. 検索ボックスにキーワードを入力して事例を検索
3. タグボタンをクリックして絞り込み
4. カードをクリックして詳細を表示
